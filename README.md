# Image Splitter

A professional-grade, SEO-optimized single-page application for splitting images into customizable parts. Built with Next.js 14, TypeScript, and TailwindCSS.

## Features

- 🖼️ Client-side image processing with no backend dependencies
- 📊 Support for all major image formats (PNG, JPEG, WebP, etc.)
- ↔️ Split images horizontally, vertically, or in a grid
- 🎛️ Customizable number of splits (2-10 parts)
- 👁️ Real-time preview of split results
- 💾 Batch download functionality
- 🌍 Bilingual interface (English/Russian)
- 🎨 Modern glassmorphism design with animated gradients
- 📱 Fully responsive design
- ♿ Accessibility features

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/image-split.git
cd image-split
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

1. Drop an image or click to upload
2. Choose split direction (horizontal/vertical/grid)
3. Adjust number of parts using the slider
4. Preview the split results in real-time
5. Download all parts or individual pieces
